Detailed Description

Класс-менеджер для работы с базой данных (реализация Singleton)

Обеспечивает:

  • Единственность экземпляра (паттерн Singleton)
  • Подключение к SQLite базе данных
  • Инициализацию структуры БД
  • Доступ к объекту QSqlDatabase для выполнения запросов
